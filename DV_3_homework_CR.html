The CPI is an index provided by the NGO Transparency International and indicates how corrupt a country is perceived to be by consumers. This variable shows the 2017 value for this index. The HDI is a well-known index for a country’s human development and composed of the live expectancy, average income and education level of a country.

Question 4

First, a plot using “lm”:

ggplot(corruptDF,
       aes(CPI2017, HDI2017)) +
  geom_point() +
  geom_smooth(method = "lm") +
  theme_minimal() +
  labs(title = "Relationship between HDI and CPI, 2017",
       x = "Corruption Perception Index (CPI)",
       y = "Human Development Index (HDI)")
## `geom_smooth()` using formula 'y ~ x'

And now using “gam”:

ggplot(corruptDF,
       aes(CPI2017, HDI2017)) +
  geom_point() +
  geom_smooth(method = "gam") +
  theme_minimal() +
  labs(title = "Relationship between HDI and CPI, 2017",
       x = "Corruption Perception Index (CPI)",
       y = "Human Development Index (HDI)")
## `geom_smooth()` using formula 'y ~ s(x, bs = "cs")'

While “lm” uses a simple linear regression model, “gam” uses a generalized additive model. Basically, “lm” will try to fit a straight line over our plot while “gam” can also show non-linear relationships. Here, I’d prefer “gam” as it shows that the correlation isn’t entirely linear, at higher CPI values (very little corruption) the HDI levels barely change.

We see that there are clear differences in the distribution of HDI and CPI values between the regions. The highest values in both indices are scored by countries in Western Europe, the lowest by those in sub-Sahara Africa. We can make these differences clearer by using facets:

ggplot(corruptDF,
       aes(CPI2017, HDI2017)) +
  geom_point() +
  geom_smooth(method = "gam") +
  theme_minimal() +
  labs(title = "Relationship between HDI and CPI, 2017",
       x = "Corruption Perception Index (CPI)",
       y = "Human Development Index (HDI)") +
  facet_wrap(~ region)
